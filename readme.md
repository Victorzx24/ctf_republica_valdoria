# Resolução de CTF — República de Valdoria

Este repositório documenta a resolução do desafio **CTF – República de Valdoria**, cujo objetivo foi analisar artefatos fictícios envolvendo **OSINT**, **reconhecimento web** e **criptografia**, a partir de documentos e superfícies digitais expostas.

Todo o conteúdo apresentado é **ficcional** e destinado exclusivamente a **fins educacionais e de treinamento**.

---

## 🎯 Objetivo do Desafio

Identificar informações relevantes a partir de:
- documentos classificados simulados;
- superfícies web acessíveis publicamente;
- artefatos criptográficos fornecidos no contexto do desafio.

A resolução exigiu análise crítica, correlação de dados e validação manual dos resultados obtidos.

---

## 🧰 Ferramentas Utilizadas

As seguintes ferramentas e recursos foram empregados durante o processo de análise:

- **Dirb**  
  Utilizado na fase de reconhecimento para identificação de diretórios e endpoints acessíveis no domínio analisado.

- **CyberChef**  
  Empregado para análise e manipulação de dados criptográficos, incluindo verificação de algoritmos, formatos e parâmetros.

- **Navegador Web (análise manual)**  
  Utilizado para validação de respostas HTTP, inspeção de conteúdo exposto e correlação contextual das informações encontradas.

---

## 🧠 Metodologia Geral

A abordagem adotada seguiu as seguintes etapas:

1. Reconhecimento inicial da superfície web;
2. Análise de documentos e artefatos fornecidos no desafio;
3. Identificação de indícios relevantes para OSINT;
4. Análise criptográfica dos dados cifrados;
5. Consolidação das informações obtidas.
